For academic work, any citation in the body of an academic report, must have a corresponding entry in the report's Reference List.

Sources which are reviewed but not directly cited in the report are then placed in a separate 'Bibliography'. There is a methodology, as some academic sources advocate, for the use of a single 'Bibliography' list (typically for final published work). For academic marking the first method is much more convenient for markers to confirm (for example) that every citation links to a Reference List item.

I noted that all of the reference list items, that appeared in your TMA01 report, were correctly presented in this TMA02 report. If you change citations and a Reference List item is no longer cited, rather than 'lose' the information source, you can move that source into a second 'Bibliography' listing. This allows you to gain 'credit' for sources that were read or reviewed but not directly cited in the main report. (another good example of the benefit of the separation of the Reference List and the Bibliography).
